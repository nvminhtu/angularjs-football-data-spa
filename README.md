# AngularJS Football Data SPA #

AngularJS SPA that consumes the freely available [football-data.org](http://football-data.org) API, and displays squad details, club fixtures, and the Premier League table for **Newcastle United FC**.

![alt-text](https://upload.wikimedia.org/wikipedia/hif/2/25/Newcastle_United_Logo.png "Newcastle United FC Crest")

- Project a work in progress, but uploaded to GitHub to utilise Git and learn AngularJS + APIs.
